## 🚀 編譯說明

### Overleaf 編譯
1. 上傳 `main_enhanced.tex` 到 Overleaf
2. 設定編譯器為 **XeLaTeX**
3. 直接編譯即可（支援中文）

### 本地編譯
```bash
# 確保安裝 TeX Live 或 MiKTeX
xelatex main_enhanced.tex
xelatex main_enhanced.tex  # 編譯兩次以正確生成目錄和引用
```

### 字體說明
- 文檔已配置字體 fallback，Overleaf 會自動處理
- 本地編譯確保有 Noto CJK 或 Fandol 字體
