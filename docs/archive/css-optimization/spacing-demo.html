📱 響應式間距建議

在不同螢幕尺寸下，間距應該適當調整以保持良好的視覺效果。

響應式間距策略

手機端 (< 768px)

  • 容器內距: --space-4
  • 組件間距: --space-4
  • 元素間距: --space-2

平板端 (≥ 768px)

  • 容器內距: --space-6
  • 組件間距: --space-6
  • 元素間距: --space-4

桌面端 (≥ 1024px)

  • 容器內距: --space-8
  • 組件間距: --space-8
  • 元素間距: --space-4

建議: 在較小螢幕上使用較小的間距，在較大螢幕上使用較大的間距，保持比例協調。
