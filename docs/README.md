## 核心概念

### 1. 智能錯誤分類
Linker CLI 採用創新的錯誤分類系統，將學習錯誤分為四大類：
- **系統性錯誤**：文法規則類，需要系統學習
- **單一性錯誤**：詞彙搭配類，需要個別記憶
- **可以更好**：表達優化類，非必要改進
- **其他錯誤**：理解偏差類，需具體分析

### 2. 掌握度管理
基於科學的遺忘曲線理論，系統會：
- 動態追蹤每個知識點的掌握程度
- 智能安排複習時間
- 根據錯誤類型調整學習策略

### 3. 模組化架構
專案採用分層架構設計：
- **展示層**：用戶介面和交互
- **業務層**：核心業務邏輯
- **服務層**：AI 服務、日誌、緩存
- **基礎層**：配置、異常、數據存儲
