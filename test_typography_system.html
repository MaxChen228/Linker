中英混排測試 (Mixed Language Test)

Linker 是一個英文翻譯練習系統，使用 Gemini API 進行智能出題和批改。 The system uses FastAPI + Jinja2 templates for SSR architecture.

系統採用雙模型架構：Gemini 2.5 Flash 負責快速出題，Gemini 2.5 Pro 負責精準批改。 This dual-model approach ensures both speed and accuracy in the learning process.

文字系統支援 responsive design，在不同設備上都能提供最佳的 user experience。 Typography variables ensure consistency across all components and pages.

中英混排測試能夠驗證字體、行高、字距等設定在多語言環境下的表現效果。 Mixed language testing validates typography settings across multilingual content.
